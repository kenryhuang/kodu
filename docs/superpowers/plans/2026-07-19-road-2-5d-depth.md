# Road 2.5D Depth Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the existing hand-painted road read as an obviously dimensional 2.5D surface through a dedicated normal map, scene lighting, and a shallow visual-only crown.

**Architecture:** Extend the deterministic Pillow asset pipeline to emit a seamless road normal map, bind it to the existing Babylon.js `StandardMaterial`, and shape the existing seven-sample road ribbon along the terrain normal. Keep physics and player traversal on the unchanged heightmap ground.

**Tech Stack:** Python 3, Pillow, TypeScript, Babylon.js, Playwright, Vite

## Global Constraints

- Preserve the current high-resolution hand-painted diffuse texture and road silhouette.
- Do not change player movement, collision, route layout, or terrain traversal height.
- Keep the road matte and avoid plastic specular highlights.
- Use the existing single road mesh; add no physics body, shadow caster, per-frame update, displacement shader, parallax mapping, or loose pebble meshes.
- The normal map must be deterministic, 1024 × 2048, RGB, vertically seamless, and generated from committed assets.
- The road crown must be symmetric, smoothly eased, and 4–6 cm above its shoulders.
- Preserve the user's unrelated working-tree changes in `public/assets/terrain/0eb25ff3-a845-49d2-a37c-fd90ef46a85b.png` and `public/assets/terrain/terrain_samples.png`.

- [ ] **Step 3: Generate a normalized tangent-space map from the final diffuse ribbon**

Add `import math`, import `RIBBON_NORMAL_NAME`, and add this function to `scripts/remaster-road-textures.py`:

```python
def make_normal_map(diffuse: Image.Image, strength: float = 2.4) -> Image.Image:
    rgba = diffuse.convert("RGBA")
    luminance = rgba.convert("L")
    low = luminance.filter(ImageFilter.GaussianBlur(radius=4.0))
    detail = ImageEnhance.Contrast(luminance).enhance(1.8)
    height = Image.blend(low, detail, 0.72)
    alpha = rgba.getchannel("A")
    width, image_height = rgba.size
    heights = list(height.getdata())
    alphas = list(alpha.getdata())
    pixels: list[tuple[int, int, int]] = []

    def sample(x: int, y: int) -> int:
        wrapped_y = y % image_height
        clamped_x = max(0, min(width - 1, x))
        return heights[wrapped_y * width + clamped_x]

    for y in range(image_height):
        for x in range(width):
            index = y * width + x
            if alphas[index] < 16:
                pixels.append((128, 128, 255))
                continue
            dx = (sample(x - 1, y) - sample(x + 1, y)) / 255 * strength
            dy = (sample(x, y - 1) - sample(x, y + 1)) / 255 * strength
            length = math.sqrt(dx * dx + dy * dy + 1)
            pixels.append((
                round((dx / length * 0.5 + 0.5) * 255),
                round((dy / length * 0.5 + 0.5) * 255),
                round((1 / length * 0.5 + 0.5) * 255),
            ))

    normal = Image.new("RGB", rgba.size)
    normal.putdata(pixels)
    normal.paste(normal.crop((0, 0, width, 1)), (0, image_height - 1))
    return normal
```

After saving the diffuse ribbon in `main`, generate and save the normal map:


```typescript
  const center = offsets[Math.floor(offsets.length / 2)];

  terrainMainRoadCrownHeight = Math.max(terrainMainRoadCrownHeight, center - shoulder);
```
